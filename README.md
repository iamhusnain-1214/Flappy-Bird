# Flappy Bird — Stranger Things Edition

A Stranger Things-inspired Flappy Bird clone built in C++ using Raylib.

## Features
- Two switchable worlds — Right Side Up and Upside Down
- Camzotz-inspired background design
- Reverse gravity mechanic that flips the entire physics system
- Real-time collision detection and score tracking

C++ · Raylib · Game Development
